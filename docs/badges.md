# Tome Badges
Badges help gamify the practice and memorization of topics. 

There are **different types of badges** available, linked to different goals. 

## Daily Practice Badge
The most common badge, it is issued when a user reaches a certain amount of *Practice Points* in the current day. 

## Perfect Week Badge
This badge is issued when a *Daily Practice Badge* has been issued **every day of the week**.
